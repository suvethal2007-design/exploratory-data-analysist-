# 📊 Exploratory Data Analysis Dashboard

## 📌 Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset using Microsoft Power BI. The dashboard provides meaningful insights into sales performance, customer demographics, product trends, and business growth through interactive visualizations.

The objective of this project is to transform raw sales data into actionable business insights that help stakeholders make data-driven decisions.

---

## 🎯 Objectives

* Analyze overall sales performance.
* Identify top-performing products and categories.
* Understand customer purchasing behavior.
* Examine sales trends over time.
* Compare sales across different cities and customer segments.
* Create an interactive dashboard for easy business analysis.

---

## 📂 Dataset Information

The dataset contains:

* Order ID
* Customer ID
* Product Details
* Category
* Quantity
* Unit Price
* Total Sales
* Gender
* Age
* City
* Order Date

---

## 📈 Dashboard Features

### KPI Cards

* Total Sales
* Total Orders
* Average Sales
* Average Customer Age

### Visualizations

* Sales by Category (Column Chart)
* Sales by City (Column Chart)
* Monthly Sales Trend (Line Chart)
* Customer Distribution by Gender (Pie Chart)
* Product-wise Sales Analysis (Bar Chart)
* Quantity vs Sales Relationship (Scatter Chart)

### Interactive Filters

* Category
* City
* Gender
* Order Date
